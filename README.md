# Lost and Found App 🧳

A web-based Lost and Found system for educational institutes, built with:

- ⚛️ React + TypeScript
- 🎨 Tailwind CSS
- ⚡ Vite
- 🔐 JWT-based authentication (coming soon)
- 📦 Hosted on GitHub

## Getting Started

```bash
npm install
npm run dev
